What this tool checks

  • Live host discovery via ARP and ICMP
  • TCP scan of common IoT ports (Telnet, FTP, HTTP, RTSP, MQTT, UPnP, CoAP, TR-069, DVR control)
  • Banner-based detection of outdated services (Boa, GoAhead, SSHv1, old OpenSSH)
  • MAC vendor fingerprinting for common IoT brands
  • Risk scoring and a remediation checklist per device

This tool only observes services that are already exposed. It does not attempt credential brute-forcing or exploit delivery.
